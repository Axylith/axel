# Support

## Getting Help

| Need | Where |
|------|-------|
| Bug report | [Open an issue](https://gitlab.com/axylith/axylith/-/issues/new) with reproduction steps |
| Feature request | [Open an issue](https://gitlab.com/axylith/axylith/-/issues/new) describing the problem it solves |
| Question about the code | Open an issue with the `question` label |
| Security vulnerability | Email security@axylith.com (see SECURITY.md) |
| Licensing or commercial inquiry | Email founders@axylith.com |
| Contributing | Read CONTRIBUTING.md |

## Before Opening an Issue

1. Search existing issues to avoid duplicates.
2. Include your OS, GPU, and driver version.
3. Include the full error output (not a screenshot — paste the text).
4. Include steps to reproduce the problem.

## What We Don't Support

- Obsidian, Jupyter, MATLAB, or any other tool. We can't help
  you migrate yet — migration tools are planned but not built.
- Custom builds with modified code. If you changed the source,
  you're on your own.
- Proprietary GPU drivers. If your driver is broken, contact
  NVIDIA/AMD/Intel.
- Windows or macOS (not yet supported, Linux only for now).

## Paid Support

Priority support is available for Team, Lab, and Enterprise tiers.
Contact founders@axylith.com for details.
